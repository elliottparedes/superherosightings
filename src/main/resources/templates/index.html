Welcome to the HERO Sightings System

This application was designed for the Hero Education and Relationship Organization (HERO) in order to keep track of any and all super hero/super villain activities. On the navbar you will see sections for Sightings, Heroes, Locations, and Organizations. To report a sighting just go to the sightings page and report it using the drop down boxes. If a hero, location, or organization needs to be added to the dropdown you can update that in the appropiate tab. Be careful deleting information. For example deleting a location will also delete any associated Sightings and Organizations associated with it. Happy hunting!
